👁️ AI Vision Accessibility Tool

A Chrome extension that uses computer vision to describe images, read text aloud, and simplify web layouts for visually impaired users.

Key Features

  • 🖼️
    Image Description
    Generates natural-language descriptions for any image on a web page using a fine-tuned BLIP model.
  • 📖
    OCR Text Extraction
    Extracts and reads embedded text from images, banners, and infographics using Tesseract.js.
  • 🗂️
    Layout Simplification
    Strips complex CSS and reformats pages into a clean, high-contrast, screen-reader-friendly layout.
  • 🔊
    Voice Navigation
    Users can navigate page elements using voice commands: "Next heading", "Read paragraph", etc.
